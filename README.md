# vlm.md

> VLM recipes for agent developers — battle-tested prompts, code, and gotchas for using Vision Language Models in agentic systems.

🌐 **[vlm.md](https://vlm.md)** · [中文](https://vlm.md/zh) · [English](https://vlm.md/en)

## What is this?

vlm.md is a community cookbook of practical VLM recipes. Each recipe covers a real engineering scenario:

- Which model to use and why
- A prompt template you can copy directly
- Working Python code (tested, not theoretical)
- Production gotchas we hit along the way

## Recipe categories

| Category | Description |
|---|---|
| Document Understanding | Invoice, PDF, form extraction |
| UI Grounding | Web UI detection, clicking, navigation |
| Chart & Table | Graph reading, table extraction |
| Computer Use | Desktop automation, screenshot-based agents |
| Multi-image Reasoning | Comparing images, sequential screenshots |
| Structured Output | Reliable JSON/structured extraction patterns |
